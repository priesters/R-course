##Part 1

p1 <- ggplot(ChickWeight, aes(x=Time, y=weight, colour=Diet)) +
    geom_point(alpha=.8) +
    geom_smooth(alpha=.2, size=0.5) +
    ggtitle("Fitted growth curve per diet")
p1
## `geom_smooth()` using method = 'loess' and formula 'y ~ x'

p1画出以times为x轴，weight为y轴，不同颜色值表示不同的对象的diet值。ase（）为映射，将数据中的各个属性映射到坐标轴或者其它几何要素上。geom_point()函数表示增加散点图图层，用于画出散点图，参数alpha是透明度，值在0到1之间。散点图的数据较多噪点较大，那么数据显示的趋势就不是十分的明显，geom—_point（）函数添加一条平滑的趋势曲线并显示标准误差，灰色部分是置信区间，参数size是曲线的粗细。 最后给图片增加标题。

p2 <- ggplot(subset(ChickWeight, Time==21), aes(x=weight, colour=Diet)) +
    geom_density(size=2) +
    ggtitle("Final weight, by diet")+theme_bw()
p2

p2画的是ChickWeight中times值为21的子集的数据。ase（）将weight设成了x轴，，不同颜色值表示不同的对象的diet值。geom_density()做出一条拟合曲线。

p3 <- ggplot(subset(ChickWeight, Time==21), aes(x=weight, fill=Diet)) +
    geom_histogram(colour="black", binwidth=50) +
    facet_grid(Diet ~ .) +
    ggtitle("Final weight, by diet") +
    theme(legend.position="none")        # No legend (redundant in this graph)
p3

geom_histogram()：直方图又称质量分布图，由一系列高度不等的纵向条纹或线段表示数据分布的情况。其中binwidth参数可以用来设置组距，通过组距调节平滑度。 p3做出分面直方图，取ChickWeight中times值为21的子集的数据。x轴是weight，通过diet的不同形成四个对象的直方图。
